# Research Notes: UX/UI Redesign

## Key Findings

- **Primary Inspiration**: Maybe Finance (open-source)
- **Design Style**: Modern fintech (Robinhood/Wealthfront aesthetic)
- **Target User**: Sophisticated but not professional investor

## Design Decisions

| Decision | Rationale | Alternatives Considered |
|----------|-----------|------------------------|
| SunnRayy Palette (Gold + Blue) | Aligned with brand identity, finance-appropriate | Original Indigo-Violet palette |
| Light backgrounds with accents | Better readability, less overwhelming | Heavy gradient backgrounds |
| 7-step Import Wizard | Matches Maybe Finance pattern, proven UX | Single-page import form |
| D3.js for charts (optional) | Better interactivity, performance | Keep Chart.js |
